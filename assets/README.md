# 客户续保预测可视化项目

## 项目概述

本项目是一个客户续保预测的可视化分析系统，使用HTML、CSS和JavaScript构建，基于ECharts库实现数据可视化。项目包含了逻辑回归和决策树两种模型的分析结果，并对新客户的续保情况进行了预测。

## 主要功能

1. **整体续保占比**：显示客户整体续保率
2. **投保人年龄分布**：展示不同年龄段的客户分布
3. **收入水平与续保结构**：分析不同收入水平的续保情况
4. **性别与续保对比**：比较不同性别的续保率
5. **职业分布Top5**：展示保单量最多的5个职业
6. **逻辑回归系数分析**：分析各特征对续保的影响程度
7. **决策树特征重要性**：展示各特征对模型预测的贡献程度
8. **新客户续保预测**：预测新客户的续保率
9. **新客户年龄分布**：分析不同年龄段新客户的续保率

## 如何在GitHub Pages上部署

1. **创建GitHub仓库**：在GitHub上创建一个新的仓库
2. **上传文件**：将本文件夹中的所有文件上传到GitHub仓库
3. **启用GitHub Pages**：
   - 进入仓库设置
   - 找到"Pages"选项
   - 选择"main"分支作为源
   - 点击"Save"按钮
4. **访问网站**：部署完成后，GitHub会提供一个URL，通过该URL可以访问可视化大屏

## 技术栈

- **前端框架**：HTML5, CSS3, JavaScript
- **可视化库**：ECharts 5
- **数据处理**：JSON

## 数据说明

项目使用的数据存储在`data.json`文件中，包括：
- 续保分布数据
- 年龄分布数据
- 逻辑回归系数数据
- 决策树特征重要性数据
- 新客户分析数据

## 本地预览

在本地环境中，可以通过以下方式预览：
1. 下载本项目的所有文件
2. 使用浏览器直接打开`index.html`文件
3. 确保`data.json`文件与`index.html`在同一目录下
