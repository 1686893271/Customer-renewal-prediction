# 客户续保预测数据字段含义

## 数据表字段含义

| 字段名 | 含义 | 详细分类 |
|-------|------|----------|
| policy_id | 保单ID，唯一标识每个保单的编号 | 数字编号 |
| age | 客户年龄 | 数值型，单位：岁 |
| gender | 客户性别 | 男、女 |
| birth_region | 客户出生地区 | 省级行政区，如湖北省、香港特别行政区等 |
| insurance_region | 保险区域 | 省级行政区，如四川省、香港特别行政区等 |
| income_level | 收入水平 | 高、中、低 |
| education_level | 教育水平 | 博士、硕士、本科、专科及以下 |
| occupation | 职业 | 医生、工程师、律师、教师、企业员工、自由职业等 |
| marital_status | 婚姻状况 | 已婚、单身、离异、丧偶等 |
| family_members | 家庭成员数量 | 数值型，单位：人 |
| policy_type | 保单类型 | 守护百分百2021、盛世福优悦版、平安盛世金越（2023）终身寿险、盛世福尊悦版等 |
| policy_term | 保单期限 | 10年、20年等 |
| premium_amount | 保费金额 | 数值型，单位：元 |
| policy_start_date | 保单开始日期 | 日期型，格式：YYYY-MM-DD |
| policy_end_date | 保单结束日期 | 日期型，格式：YYYY-MM-DD |
| claim_history | 理赔历史 | 是、否 |
| renewal | 续保情况 | Yes（是）、No（否） |

| 保单类型 | 编码 | 产品特点 |
|---------|------|----------|
| 守护百分百2021 | 0 | 综合保障型产品 |
| 盛世福优悦版 | 1 | 重疾保障型产品 |
| 平安盛世金越（2023）终身寿险 | 2 | 终身寿险产品 |
| 盛世福尊悦版 | 3 | 高端保障型产品 |
| 其他类型1 | 4 | 保障型产品 |
| 其他类型2 | 5 | 储蓄型产品 |
| 其他类型3 | 6 | 健康型产品 |
| 其他类型4 | 7 | 意外型产品 |
| 其他类型5 | 8 | 综合型产品 |

|---------|------|----------|

| 中 | 1 | 中等收入群体 |
